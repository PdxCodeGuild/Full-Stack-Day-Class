### Running Doctests

First, [follow the instructions](/notes/django-init.md) I've written up to add a `tests.py` to your Django app.
Then, you can run tests through Django.

* Script: `manage.py`
* Script parameters: `test`

### Debug Run Server

* Script: `manage.py`
* Script parameters: `runserver`
